## 一、财务分析核心计算

### 1. NPV (净现值)
```
NPV = Σ (现金流 / (1 + 折现率)^年份)
```
- 将未来现金流折算成今天的价值
- 折现率通常用 10%（机会成本）
- NPV > 0 表示可行

### 2. IRR (内部收益率)
- 使 NPV = 0 的折现率
- IRR > 目标回报率 则可行

### 3. 投资回收期
```
回收期 = 初始投资 / 年现金流
```
- 越短越好

### 4. ROI (投资回报率)
```
ROI = (总收益 - 投资成本) / 投资成本 × 100%
```

## 二、创业分析专家技能 (startup-analyst)

### 核心专长

#### 1. 市场规模计算
- **TAM (总潜在市场)**: 整体市场需求
- **SAM (可服务市场)**: 你能服务的市场
- **SOM (可获得市场)**: 你能实际占领的

**计算方法**:
- 自上而下 (Top-down): 从大市场往小了算
- 自下而上 (Bottom-up): 从具体客户往大了算
- 价值理论: 新市场类别

#### 2. 财务建模
- 基于队列的收入预测
- 单位经济学 (CAC, LTV, 回收期)
- 3-5年财务模型
- 现金流预测和 runway 分析
- 烧钱率和效率指标

#### 3. 竞争分析框架
- 波特五力模型
- 蓝海战略
- 竞争定位和差异化
- 市场格局 mapping
- 可持续竞争优势评估

#### 4. 团队和组织规划
- 按阶段招聘计划 (pre-seed, seed, Series A)
- 薪酬基准和股权分配
- 组织设计和汇报结构

#### 5. 创业指标 (KPIs)

| 指标 | 公式 | 目标 |
|------|------|------|
| CAC | 营销费用 / 新客户数 | 越低越好 |
| LTV | ARPU × 毛利率 × (1/流失率) | CAC 的 3倍 |
| LTV:CAC | LTV / CAC | ≥ 3:1 |
| CAC 回收期 | CAC / (ARPU × 毛利率) | < 12个月 |
| 烧钱率 | 每月现金消耗 | 越低越好 |
| runway | 现金 / 烧钱率 | > 18个月 |

## 三、Go-to-Market (GTM) 规划

### GTM 计划核心要素

#### 1. 目标客户和细分市场
- 主要客户画像
- 优先进入的海滩头市场
- 购买旅程: 认知 → 考虑 → 决策

#### 2. 渠道策略

**有机渠道** (低成本，高投入):
- SEO / 内容营销
- 社交媒体
- 社区运营
- 推荐/口碑

**付费渠道** (高成本，快结果):
- Google 搜索广告
- LinkedIn / Facebook 广告
- 再营销广告

**直销** (高触达):
- 冷外展
- 合作伙伴/集成

**产品驱动** (内置于产品):
- 免费版/免费试用
- 病毒式/推荐计划

#### 3. 90天执行路线图

**第1月: 基础和启动**
- 周1: 网站上线、设置分析追踪
- 周2-4: 启动获客渠道

**第2月: 增长和优化**
- 启动付费广告
- 发布 SEO 内容
- 冷外展

**第3月: 规模和迭代**
- 增加付费预算
- 启动推荐计划
- 达成第一个合作

#### 4. 预算分配示例

| 类别 | 月预算 | 占比 |
|------|--------|------|
| 付费广告 | ¥5000 | 50% |
| 内容/SEO | ¥2000 | 20% |
| 工具/软件 | ¥1000 | 10% |
| 活动/合作 | ¥2000 | 20% |

### 按融资阶段
