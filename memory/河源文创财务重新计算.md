## 一、现实情况分析

### 1.1 景区客流≠进店客流

| 数据 | 说明 |
|------|------|
| 万绿湖年客流 45万 | 这是总客流，不是天天有 |
| 节假日占 30% | 元旦/春节/五一/国庆是高峰 |
| 平时占 70% | 平时客流很少 |
| 路过店门口 | 不一定进店，预计 20-30% |
| 进店→购买 | 不一定买，预计 15-25% |

### 1.2 现实转化漏斗

```
景区游客 100人
    ↓ 经过店铺 (30%)
进店 30人
    ↓ 看看就走 (60%)
真正选购 12人
    ↓ 嫌贵/犹豫 (40%)
成交 7人
```

**最终转化率：7%**（而不是之前假设的30%）

## 三、问题：收入太低！

### 3.1 为什么？

| 问题 | 分析 |
|------|------|
| 客流被高估 | 景区≠店铺客流 |
| 转化率太高 | 30%转化不可能 |
| 客单价太高 | 游客精打细算 |

### 3.2 实际情况

河源是**四五线城市**，消费能力有限：
- 游客主要消费：吃饭、住宿、交通
- 纪念品预算：人均 20-30 元
- 网上更便宜：淘宝同款更便宜

### 6.1 真实收入预测

| 模式 | 月营收 | 月成本 | 月利润 |
|------|--------|--------|--------|
| 摆摊（保守） | 2400元 | 2000元 | +400元 |

### 6.2 问题所在

**小店模式在保守情况下会亏损！**

原因：
- 租金 + 人工 = 4800元/月
- 需要 5000+ 元营收才能盈亏平衡
