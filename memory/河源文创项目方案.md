# 河源文创产品店 整体项目方案

## 一、项目概述

**项目名称**：河源特色文创产品店  
**定位**：河源本地文创+农特产品融合店  
**模式**：线下门店 + 线上微店 + 旅游渠道合作

---

## 二、市场分析

### 2.1 行业背景
- 文创产业蓬勃发展，国家大力支持
- 旅游消费升级，游客追求特色纪念品
- 乡村振兴政策支持农产品品牌化

### 2.2 河源优势
| 资源 | 价值 |
|------|------|
| 万绿湖 | 年游客300万+ |
| 客家文化 | 深厚底蕴，独特IP |
| 非遗项目 | 香火龙、木偶戏、土陶等 |
| 恐龙IP | "恐龙之乡"独有资源 |
| 农特产品 | 鹰嘴桃、茶叶、柑橘等 |

### 2.3 目标客群
1. **游客**（占比60%）- 购买纪念品、伴手礼
2. **本地居民**（占比30%）- 送礼、日常消费
3. **电商客群**（占比10%）- 外地也可购买

## 三、产品规划

### 3.1 四大产品线

| 产品线 | 示例 | 定价区间 |
|--------|------|----------|
| **IP文创** | 恐龙钥匙扣、冰箱贴、帆布包 | 19-99元 |
| **非遗手作** | 墩头蓝杯垫、陶艺、竹编 | 39-199元 |
| **客家礼盒** | 茶叶+糕点组合、特产礼盒 | 68-288元 |
| **定制文创** | 照片打印、定制马克杯 | 49-159元 |

### 3.2 产品策略
- **引流款**：低价文创（19-39元）
- **利润款**：礼盒组合（100-200元）
- **高端款**：定制/限量款（200+元）

## 七、选址建议

1. **🔥 必选**：万绿湖景区门口/停车场旁
