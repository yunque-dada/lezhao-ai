Scratch 图形化编程
6-10岁64课时20+项目

Scratch是由MIT麻省理工学院开发的少儿编程入门工具，通过拖拽积木块的方式学习编程逻辑。孩子不需要输入代码，只需要像拼图一样组合不同功能的积木，就能创造出动画、游戏和故事。这是一种非常适合培养计算思维的方式，让孩子在玩耍中掌握编程的基本概念。

📖 课程大纲

第一阶段：基础入门（16课时）
  • Scratch界面认识和基本操作
  • 角色的创建和编辑
  • 背景的添加和切换
  • 运动模块：移动、旋转、跳跃
  • 外观模块：造型切换、大小变化
  • 声音模块：添加背景音乐和音效
第二阶段：逻辑进阶（16课时）
  • 事件模块：按键触发、点击触发
  • 条件判断：如果...那么...
  • 循环结构：重复执行、等待
  • 变量：什么是变量、如何使用
  • 运算：数学运算、逻辑运算
  • 侦测模块：碰撞检测、距离检测
第三阶段：技能提升（16课时）
  • 列表：存储多个数据
  • 克隆：创建多个相同角色
  • 自定义积木：封装重复代码
  • 坐标系统：XY坐标精确定位
  • 消息广播：角色间通信
  • 计时器和倒计时
第四阶段：项目实战（16课时）
  • 完整游戏项目开发
  • 动画短片制作
  • 互动故事创作
  • 多人游戏设计
  • 作品展示和分享
  • 参加Scratch比赛

Python 编程
10-14岁80课时15个项目

Python是当前最受欢迎的编程语言之一，语法简洁易懂，应用广泛。从数据分析到人工智能，从网站开发到游戏制作，Python都能胜任。本课程让孩子从零开始，逐步掌握Python的核心技能，能够独立编写实用程序。同时，Python也是参加信息学竞赛的绝佳选择。

📖 课程大纲

第一阶段：Python入门（20课时）
  • Python简介和应用领域
  • 开发环境搭建
  • 第一个Python程序
  • 变量和数据类型
  • 输入输出和格式化
  • 算术运算符和表达式
第二阶段：程序控制（20课时）
  • 布尔类型和比较运算
  • if条件判断语句
  • if-elif-else多分支
  • while循环和循环嵌套
  • for循环和range函数
  • break和continue控制
第三阶段：函数和模块（20课时）
  • 函数的定义和调用
  • 参数传递和返回值
  • 变量的作用域
  • 模块和标准库
  • 常用模块：random/time/math
  • 第三方库安装和使用
第四阶段：数据结构和项目（20课时）
  • 列表List的增删改查
  • 字典Dictionary使用
  • 集合和元组
  • 文件读写操作
  • 异常处理
  • 综合项目开发

C++ 信息学奥赛
12-18岁120+课时CSP/NOIP

C++是信息学奥林匹克竞赛（NOIP/NOI）的指定语言，也是计算机奥林匹克竞赛的国际通用语言。本课程系统学习C++语言和算法数据结构，为参加CSP-S、NOIP省赛、NOI全国赛做准备。获得竞赛奖项可以为升学提供有力帮助。

📖 课程大纲

C++基础（30课时）
  • C++简介和开发环境
  • 基本数据类型和变量
  • 算术、关系、逻辑运算符
  • if-else条件判断
  • for/while循环
  • 数组和字符串
基础算法（30课时）
  • 枚举和模拟
  • 排序算法
  • 二分查找和分治
  • 高精度计算
  • 递推和递归
  • 搜索入门：DFS/BFS
数据结构（30课时）
  • 栈和队列
  • 链表和链表操作
  • 树和二叉树基础
  • 堆和优先队列
  • 并查集
  • 哈希表
高级算法（30课时）
  • 动态规划基础
  • 背包问题
  • 最长公共子序列
  • 最短路径：Dijkstra
  • 最小生成树
  • 图论综合训练
